# GreenSpot 테스트 케이스

| 항목 | 내용 |
| --- | --- |
| 버전 | 2.0 |
| 작성일 | 2026-07-09 |
| 대상 | GreenSpot 프론트 UI + 백엔드 API (연동 포함) |
| 기준 | 기능명세서, api.md |
| 체크 | `V` 통과 / `X` 실패 / `-` 스킵 |

### 사전 조건

| # | 내용 |
| --- | --- |
| 1 | 백엔드 기동 (`http://localhost:8000`) |
| 2 | 프론트 기동 (예: `http://localhost:5173`) |
| 3 | (라이브 테스트) `.env` 에 `VWORLD_API_KEY` 등 설정 |
| 4 | 시드 데이터 적재 (`python -m scripts.seed`) |

### 공통 데이터

| 이름 | 예시 |
| --- | --- |
| 신규 이메일 | `test_YYYYMMDD@example.com` (매번 고유) |
| 비밀번호 | `testpass1` (6자 이상) |
| 이름 | `테스터` |
| 검색 지역 | `금천구` / `용산구` |
